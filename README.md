# Normas-Construccion-Neuquen

Repositorio de trabajo para pensar y construir, de forma gradual y sostenible, un sistema de gestion y consulta de normas de construccion para Neuquen y su area de influencia.

## Contenido actual
- `docs/hoja-de-ruta-sistema-normativo.md`: hoja de ruta metodologica para construir el sistema paso a paso, priorizando fuentes oficiales, trazabilidad y sostenibilidad.
- `docs/notas-arquitectura-abierta.md`: opciones de implementacion y criterios de decision para una arquitectura futura, sin fijar todavia un stack o solucion final.

## Enfoque actual
Por ahora el repositorio no define una arquitectura cerrada. La prioridad es:
- consolidar el metodo
- ordenar fuentes y criterios
- dejar trazabilidad
- postergar la decision tecnica hasta tener mas claridad real sobre el uso del sistema

## Proximo criterio de avance
Antes de decidir implementacion, deberian validarse estos puntos:
- alcance inicial del sistema
- fuentes oficiales prioritarias
- estructura documental base
- modelo de base maestra
- tipo de consultas que realmente hacen falta

## Objetivo del repo
Que funcione como referencia viva para ordenar decisiones y avanzar sin improvisar ni sobredisenar la solucion demasiado pronto.
